# ft_server

Summary: This is a System Administration subject. You will discover Docker and you
will set up your first web server.

> CMD build:<br>
 > `$> docker build .`<br>
 > CMD run:<br>
 > `$> docker run -it -p 80:80 -p 443:443 <container_ID_from_past_command>/bin/bash`<br>
